# anomalyAI
Network Anomaly Detection//Детектор Аномалий Трафика

Разработанная система представляет собой программный комплекс для обнаружения аномалий в сетевом трафике, реализованный на языке Python. Решение состоит из трех основных модулей:
1. Модуль подготовки данных (data.py)---обеспечивает загрузку, предобработку и нормализацию данных из различных источников
2. Модуль обучения модели (model.py)---реализует нейросетевую архитектуру автоэнкодера для выявления аномалий
3. Модуль детектирования (detector.py)---осуществляет анализ трафика в реальном времени

Система может функционировать в двух режимах:
1. Автономный режим - генерация синтетических данных для тестирования
2. Режим реальных данных - загрузка и анализ реальных датасетов (NSL-KDD, CIC-IDS2017 и др.)
