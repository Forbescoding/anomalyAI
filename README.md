# anomalyAI
Network Anomaly Detection//Детектор Аномалий Трафика


#Разработанная система представляет собой программный комплекс для обнаружения аномалий в сетевом трафике, реализованный на языке Python. Решение состоит из трех основных модулей:
1. Модуль подготовки данных (data.py)---обеспечивает загрузку, предобработку и нормализацию данных из различных источников
2. Модуль обучения модели (model.py)---реализует нейросетевую архитектуру автоэнкодера для выявления аномалий
3. Модуль детектирования (detector.py)---осуществляет анализ трафика в реальном времени


Система может функционировать в двух режимах:
1. Автономный режим - генерация синтетических данных для тестирования
2. Режим реальных данных - загрузка и анализ реальных датасетов (NSL-KDD, CIC-IDS2017 и др.)


Пошаговый запуск программы:
1. Запустите программу data.py
   
   #сгенерируется датасет, будут разделены выборки, сохранятся следующие файлы:
   
   X_train.npy, X_test.npy - признаки

   y_train.npy, y_test.npy - метки

   feature_names.pkl - названия признаков

   scaler.pkl - нормализатор

   label_encoders.pkl - кодировщики
   
2. Запустите model.py
   
   #загрузятся подготовленные данные, создастся и обучится нейронная сеть (автоэнкодер), автоматически определится порог для обнаружения аномалий, оценится качество модели на тестовых данных, сохранятся файлы:

   anomaly_detector.h5 - обученная модель

   anomaly_threshold.pkl - порог аномалий

   confusion_matrix.png - матрица ошибок

   error_distribution.png - распределение ошибок

   training_history.png - график обучения
   
3. Запустите detector.py

   #загрузится обученная модель,   начнется симуляция сетевого трафика, в реальном времени будет определяться, является ли трафик аномальным. Программа выводит статистику обнаружений
  
4. Используйте main.py для использования всех скриптов в одной программе
