# anomalyAI
Network Anomaly Detection//Детектор Аномалий Трафика

Разработанная система представляет собой программный комплекс для обнаружения аномалий в сетевом трафике, реализованный на языке Python. Решение состоит из трех основных модулей:
Модуль подготовки данных (data.py)---обеспечивает загрузку, предобработку и нормализацию данных из различных источников
Модуль обучения модели (model.py)---реализует нейросетевую архитектуру автоэнкодера для выявления аномалий
Модуль детектирования (detector.py)---осуществляет анализ трафика в реальном времени
Система может функционировать в двух режимах:
Автономный режим - генерация синтетических данных для тестирования
Режим реальных данных - загрузка и анализ реальных датасетов (NSL-KDD, CIC-IDS2017 и др.)
